
<h1 align="center">CellsDB</h1>
<p align="center">A Local Database Solution Inspired by Cellular Structures</p>
<p align="center">English | <a href="README.zh-CN.md">中文</a></p>

under development

Cells-DB is a lightweight database framework built on PouchDB and IndexedDB, designed to store data as “cells” in the browser. Each cell is a flexible unit that can be interconnected to form powerful data relationships, enabling seamless scenarios like lists, details, tags, folders, and more. With a simple and intuitive API, Cells-DB empowers developers to manage complex data structures locally, just like working with a full-featured backend.
